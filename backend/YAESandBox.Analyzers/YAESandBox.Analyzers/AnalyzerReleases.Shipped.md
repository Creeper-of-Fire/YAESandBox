## Release 1.0

### New Rules

| Rule ID | Category | Severity | Notes                                                |
|---------|----------|----------|------------------------------------------------------|
| RU0001  | Usage    | Error    | `Result` 或其派生类型的返回值必须被使用。这可以防止意外忽略方法执行结果，尤其是其中的错误信息。 |
| EU0001  | Usage    | Error    | 在 catch 块中声明的异常变量未被传递或完整传递。只使用其属性（如.Message）会丢失堆栈信息。 |
